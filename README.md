# Expense Tracker 💰

A simple Expense Tracker application built using Python and MySQL to manage daily expenses.

---

## 🚀 Features
- Add new expenses  
- View all expenses  
- Delete expenses  
- Store data using MySQL database  

---

## 🛠️ Technologies Used
- Python  
- MySQL  

---

## ⚙️ How to Run

1. Install Python  
2. Install required packages:
   pip install -r requirements.txt  

3. Setup MySQL database  
4. Run the project:
   python main.py
